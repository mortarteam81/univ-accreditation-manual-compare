## 4. 구현 내용

### 4.1 CSS (약 80줄 추가)
- `.cmp-nav`, `.cmp-body`, `.cmp-table` — 법령 대비표 2-column 레이아웃
- `.row-title` (제목행) / `.row-meta` (매핑정보행) / `.row-kept` (유지) / `.row-modified` (변경)
- `.cell-new` (신설 셀) / `.cell-del` (삭제 셀)
- `.w-ins` (녹색 배경) / `.w-del` (빨강 취소선) / `.w-eq` (기본)
- `.ikey` (monospace 번호 태그) / `.subfield-tag` (보라 태그)
- `.cmp-nav-item.selected` — 왼쪽 네비 선택 강조
- `@media print` — 네비/헤더 숨김 (DOCX Export 전까지의 대체)

### 4.2 JavaScript 함수 (약 270줄 추가)
| 함수 | 역할 |
|------|------|
| `tokenize(s)` | 공백 기준 어절 분리 (공백 토큰도 보존) |
| `diffWords(a, b)` | LCS DP → `[{op: 'eq'|'ins'|'del', tok}]` |
| `renderLeftCellHtml(ops)` | 좌측(3주기) 셀: `ins` 제외, `del` 은 취소선 |
| `renderRightCellHtml(ops)` | 우측(4주기) 셀: `del` 제외, `ins` 는 녹색 |
| `buildCompareGroup(crit4)` | 4주기 준거 → `{crit4, sources3, mapping_type, mapping_note}` |
| `collectItems(critIds)` | 여러 3주기 items 병합 |
| `alignItemsByKey(items3, items4)` | `(subfield + item_key)` 복합키로 행 정렬 |
| `renderCompareTable(group)` | 2-column 표 생성 (제목행/메타행/items행) |
| `renderCompareNav()` | 좌 네비 렌더 + 필터 (편람/검색/신설변경만) |
| `renderCompareBody()` | 우 본문 렌더 (헤더 + 표) |
| `selectCompareCrit(cid)` | Tab 3 선택 상태 관리 |

### 4.3 상태·이벤트 연동
- `state.ui.compareSelected` — Tab 3 선택된 4주기 준거 id (기본 null)
- `state.ui.changedOnly` — 신설·변경만 보기 토글 (기본 true)
- `switchTab('compare')` — Tab 3 전환 시 nav + body 렌더
- `#changed-only-toggle` change → `renderCompareNav()`
- `#part-select` change → `compareSelected` 초기화 + nav 재렌더
- `#search-box` input → nav 도 함께 갱신
- `reset-btn` → compareSelected 도 초기화
- 단축키 `3` → Tab 3 전환, `Esc` → compareSelected 도 해제

## 6. 한계·주의

1. **DOCX Export 미포함** — 다음 세션 구현. 현재는 브라우저 인쇄(⌘+P) 로 종이/PDF 변환 가능 (@media print 적용됨).
2. **어절 diff 한계** — "운영한다" → "이행한다" 처럼 한 어절이 통째로 교체되면 둘 다 하이라이트됨 (실무상 문제 없음)
3. **공백 토큰 하이라이트** — 공백도 토큰이라 어절 사이 공백이 `<span class="w-ins">` 로 감싸져 미세하게 녹색 sliver 가 보일 수 있음 (padding 0 2px 로 최소화)
4. **N:1 통합 시 items 매칭** — 3주기 준거 2개가 같은 item_key 를 쓰면 첫 번째만 4주기와 매칭되고 나머지는 "3주기 단독 행"으로 표시. 실제 데이터에서는 item_key 가 대부분 준거별로 다르므로 문제 없음.
5. **신설·변경 필터의 기준** — `mapping_type` 의 base 가 "유지"인 것만 숨김. "유지(확장)" 같은 변형은 변경으로 표시됨 (의도)
