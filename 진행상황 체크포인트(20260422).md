## 1. 확정된 전체 설계

- 주요점검사항 파일 mapping_type 체계 통일 (원본은 `mapping_type_original` 필드에 보존)

- 계층형 트리 뼈대: `Cycle → Part → Criterion → Item`

- 임베딩: **Sentence-Transformers 다국어 모델 주검색 + TF-IDF 보조점수 (하이브리드)**
  - 모델: `paraphrase-multilingual-MiniLM-L12-v2` (로컬 다운로드 400MB)
  - 대상: criterion.content, item.content
- 산출물: `graph_nodes.json`, `graph_edges.json`, `embeddings.json`, `graph_meta.json`

### Phase 3. 대시보드 HTML 프로토타입 — 🔜 다음 세션
단일 HTML (D3.js + Tailwind CDN, 로컬 JSON 로드)

**화면 구성 (사용자 확정)**
- **메인 탭**: 매핑 탐색 (좌 3주기 / 중 매핑 화살표 + 변경 유형 색상 / 우 4주기)
- **클릭 시 우측 패널 또는 하단 탭**: 변경 영향도
- **별도 탭**: 전체 구조 비교 (Sankey/Chord)

## 2. Phase 1 검수 결과 요약

### 자동 검수 통계
- 1차 audit 결과: 18건 이상 후보
- 교차 참조 자동 검증 후 진짜 이상: **6건 (4개 group)**
- 주요점검사항 mapping_type 체계 통일 대상: 16건
- **최종 수정 적용: 22건**

### 적용된 수정 (22건)
| 파트 | gid | 필드 | 변경 |
|------|-----|------|------|
| 주요점검사항 | 1 | mapping_note | `'장단기'→'중장기'`, `'특성화계획 포함' 삭제` 문구 제거 (items에 없는 개요/보고서 차원 변경) |
| 주요점검사항 | 5 | mapping_type | `통합 (N:1)` → `이동·통합 (N:1)` |
| 주요점검사항 | 6 | mapping_type | `통합 (N:1)` → `통합·이동 (N:1)` |
| 주요점검사항 | 7-15, 17-18, 22-24 (11건) | mapping_type | `변경됨 (1:1)` → `수정 (1:1)` |
| 주요점검사항 | 16 | mapping_type | `통합 (N:1)` → `통합·이동 (N:1)` |
| 주요점검사항 | 19 | mapping_type | `변경됨 (1:1)` → `유지 (1:1)` (items 실질 동일, 띄어쓰기만 차이) |
| 주요점검사항 | 20 | mapping_type | `통합 (N:1)` → `통합·분리 (N:1)` |
| 주요점검사항 | 21 | mapping_type | `이동·변경됨 (1:1)` → `통합·이동 (N:1)` (cycle_3=4개) |
| 유의사항및관련정책 | 7 | mapping_note | `'교과+비교과'→'교과과정 중심'` → `'교과과정+비교과과정 구분'→'교과과정 중심'` |

### 이상 아님 (false positive)
- 근거자료 gid=18: `·`(U+00B7) vs `･`(U+FF65) 문자 유형만 다름 → mapping_note 유지
