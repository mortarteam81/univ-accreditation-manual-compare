## 5. 주요 기능

### Tab 1 — 매핑 탐색
- 좌(3주기) / 중(SVG 베지어 매핑선) / 우(4주기) 고정 3-column
- 변경유형 색상: 유지(회색) · 수정(파랑) · 통합(주황) · 분할(보라) · 이동(청록) · 복합형(금)
- MERGED_WITH: 좌측 내부 회색 점선 곡선
- SIMILAR_TO: 선택된 준거에 한해서만 연한 점선으로 추가 (노이즈 방지)
- 좌·우 리스트 클릭 → 선택 강조 + 관련 준거 초록 배경 + 나머지 dim

### Tab 2 — 영향도 (5개 섹션)
1. ① 선택 준거 요약 (타이틀, 그룹, 변경유형 뱃지)
2. ② 확정 매핑 — MAPS_TO(3→4 / 4←3) + 같은 그룹의 MERGED_WITH 자매 · 신설/삭제 후보 경고
3. ③ Item 수준 — 하위 항목 본문 + mapping_note · evidence 는 subfield 별 그룹핑 · ITEM_MAPS_TO 자동 파싱 결과
4. ④ Cross-part — 같은 주기 내 다른 편람 섹션과의 SIMILAR_TO 연결 (5개 편람 매핑 지도)
5. ⑤ SIMILAR_TO 임베딩 후보 — 확정/신규후보 구분 · cross-cycle 우선 · 신규후보는 노란배경 강조

### 전역 컨트롤
- 편람 드롭다운 (5종)
- SIMILAR_TO threshold 슬라이더 (0.55 ~ 1.0, 기본 0.75) — SVG + Tab 2 에 동시 반영
- same-cycle SIMILAR_TO 토글 (기본 OFF — cross-cycle 만)
- 제목/ID 검색
- 초기화 버튼
- 단축키: `1`·`2` 탭 전환, `Esc` 선택 해제
